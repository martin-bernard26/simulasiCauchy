Variabel & Tipe Data
📦

Deklarasi Variabel

Cara menyimpan data sementara dalam program

Pemula

Variabel adalah tempat penyimpanan data di memori. Gunakan Dim untuk mendeklarasikan variabel. Selalu tambahkan Option Explicit di atas modul agar semua variabel wajib dideklarasikan terlebih dahulu.

StringIntegerLongDoubleBooleanDateVariant 
Option Explicit

Sub ContohVariabel()
    ' Deklarasi berbagai tipe data
    Dim namaSiswa  As String
    Dim nilaiUjian As Integer   ' -32768 s/d 32767
    Dim totalData  As Long      ' angka besar
    Dim rata2      As Double    ' bilangan desimal
    Dim lulus      As Boolean
    Dim tglLahir   As Date

    ' Mengisi nilai
    namaSiswa  = "Budi Santoso"
    nilaiUjian = 85
    rata2      = 87.5
    lulus      = True
    tglLahir   = #01/15/2000#

    ' Tampilkan ke sel
    Range("A1").Value = namaSiswa
    Range("B1").Value = nilaiUjian
End Sub
💡Gunakan Long bukan Integer untuk angka baris Excel, karena Excel bisa memiliki lebih dari 32.767 baris. Double cocok untuk perhitungan keuangan dengan desimal.

Kondisi & Logika
🔀

If - ElseIf - Else

Membuat keputusan berdasarkan kondisi

Pemula

Percabangan memungkinkan program mengeksekusi kode yang berbeda berdasarkan kondisi tertentu. VBA mendukung If...Then...Else dan Select Case untuk kondisi yang lebih kompleks.

Sub CekNilai()
    Dim nilai As Integer
    Dim grade As String
    nilai = Range("A1").Value

    ' Menggunakan If - ElseIf - Else
    If nilai >= 90 Then
        grade = "A - Sangat Baik"
    ElseIf nilai >= 80 Then
        grade = "B - Baik"
    ElseIf nilai >= 70 Then
        grade = "C - Cukup"
    Else
        grade = "D - Tidak Lulus"
    End If

    ' Menggunakan Select Case (lebih bersih)
    Select Case nilai
        Case 90 To 100 : grade = "A"
        Case 80 To 89  : grade = "B"
        Case Is >= 70   : grade = "C"
        Case Else       : grade = "D"
    End Select

    Range("B1").Value = grade
End Sub
💡Select Case lebih mudah dibaca daripada banyak ElseIf. Gunakan And / Or / Not untuk menggabungkan kondisi logis.
Perulangan (Looping)
🔄

For, For Each, Do While, Do Until

Mengulang eksekusi kode secara efisien

Pemula

Perulangan (loop) digunakan untuk mengeksekusi blok kode berulang kali. VBA menyediakan beberapa jenis loop untuk berbagai kebutuhan.

Sub ContohLoop()
    Dim i As Long
    Dim sel As Range
    Dim total As Double

    ' For Next — jumlah iterasi diketahui
    For i = 1 To 10
        Cells(i, 1).Value = i * i  ' isi kuadrat
    Next i

    ' For Each — iterasi setiap objek dalam koleksi
    For Each sel In Range("A1:A10")
        total = total + sel.Value
        If sel.Value < 0 Then sel.Interior.Color = vbRed
    Next sel

    ' Do While — ulangi selama kondisi True
    i = 1
    Do While Cells(i, 1).Value <> ""
        Cells(i, 2).Value = Cells(i, 1).Value * 2
        i = i + 1
    Loop

    ' Do Until — ulangi sampai kondisi True
    i = 1
    Do Until IsEmpty(Cells(i, 1))
        i = i + 1
    Loop
    MsgBox "Baris terakhir: " & (i - 1)
End Sub
💡Gunakan Exit For atau Exit Do untuk keluar lebih awal dari loop. Hindari loop tanpa batas (infinite loop) yang bisa membekukan Excel.
Sub & Function
⚙️

Prosedur Sub dan Function

Memecah kode menjadi blok yang dapat digunakan ulang

Menengah

Sub menjalankan serangkaian perintah tanpa mengembalikan nilai. Function seperti Sub, tetapi mengembalikan nilai — bisa digunakan langsung di sel Excel seperti rumus biasa.

' === SUB: tidak mengembalikan nilai ===
Sub FormatHeader(barisHeader As Long)
    With Rows(barisHeader)
        .Font.Bold      = True
        .Font.Size      = 12
        .Interior.Color = RGB(0, 112, 54)
        .Font.Color     = vbWhite
    End With
End Sub

' === FUNCTION: mengembalikan nilai ===
Function HitungDiskon(harga As Double, _
                       persen As Double) As Double
    HitungDiskon = harga * ((100 - persen) / 100)
End Function

' === Cara memanggilnya ===
Sub Utama()
    FormatHeader 1  ' panggil Sub

    Dim hargaFinal As Double
    hargaFinal = HitungDiskon(150000, 20)
    MsgBox "Harga setelah diskon: Rp " & hargaFinal

    ' Function juga bisa dipakai di sel Excel:
    ' =HitungDiskon(A2, 20)
End Sub
💡Gunakan ByVal jika tidak ingin nilai asli berubah, dan ByRef (default) jika ingin fungsi bisa mengubah nilai variabel yang dikirim.
Manipulasi Range & Cells
📊

Bekerja dengan Range, Cells, dan Worksheet

Inti dari otomasi Excel

Menengah

Objek Range dan Cells adalah kunci untuk membaca dan menulis data di spreadsheet. Selalu referensikan Worksheet secara eksplisit untuk kode yang lebih aman.

Sub ManipulasiRange()
    Dim ws     As Worksheet
    Dim rng    As Range
    Dim lastRow As Long

    Set ws = ThisWorkbook.Sheets("Data")

    ' Cari baris terakhir yang berisi data
    lastRow = ws.Cells(ws.Rows.Count, "A").End(xlUp).Row

    ' Tulis nilai ke sel
    ws.Range("A1").Value         = "Nama Produk"
    ws.Cells(1, 2).Value          = "Harga"

    ' Baca nilai dari sel
    Dim nilaiA1 As String
    nilaiA1 = ws.Range("A1").Value

    ' Format range sekaligus
    With ws.Range("A1:E1")
        .Font.Bold         = True
        .Interior.Color    = RGB(0, 112, 54)
        .Font.Color        = vbWhite
        .HorizontalAlignment = xlCenter
    End With

    ' Copy paste dengan cara efisien
    ws.Range("A2:A" & lastRow).Copy _
        Destination:=ws.Range("F2")

    ' Hapus isi tanpa hapus format
    ws.Range("G2:G" & lastRow).ClearContents
End Sub
💡Selalu gunakan ws.Cells(...) bukan Cells(...) saja agar kode selalu merujuk ke sheet yang tepat, bukan sheet yang sedang aktif.
Event & Worksheet
⚡

Event Handler

Kode yang berjalan otomatis saat ada aksi tertentu

Menengah

Event handler adalah prosedur yang berjalan otomatis saat sesuatu terjadi di Excel — seperti saat sel berubah, sheet diaktifkan, atau workbook dibuka. Ditempatkan di modul objek (bukan modul biasa).

' === Di modul Sheet (klik kanan sheet → View Code) ===

' Berjalan saat nilai sel berubah
Private Sub Worksheet_Change(ByVal Target As Range)
    ' Cek apakah perubahan di kolom A
    If Not Intersect(Target, Me.Columns("A")) _
        Is Nothing Then

        Application.EnableEvents = False  ' hindari loop
        Target.Value = UCase(Target.Value)  ' huruf kapital
        Application.EnableEvents = True
    End If
End Sub

' Berjalan saat sheet diaktifkan
Private Sub Worksheet_Activate()
    MsgBox "Selamat datang di sheet " & Me.Name
End Sub

' === Di modul ThisWorkbook ===

' Berjalan saat file dibuka
Private Sub Workbook_Open()
    Sheets("Dashboard").Activate
    MsgBox "File dibuka: " & Now()
End Sub

' Berjalan saat file akan disimpan
Private Sub Workbook_BeforeSave(ByVal _
    SaveAsUI As Boolean, Cancel As Boolean)
    Range("A1").Value = "Terakhir disimpan: " & Now()
End Sub
⚠️Selalu set Application.EnableEvents = False sebelum mengubah sel di dalam Worksheet_Change, lalu kembalikan ke True, agar tidak terjadi rekursi tak terbatas.
Array
🗂️

Array Statis & Dinamis

Menyimpan dan memproses banyak data sekaligus

Lanjut

Array memungkinkan penyimpanan banyak nilai dalam satu variabel. Memuat data ke array lalu menulisnya sekaligus jauh lebih cepat daripada baca-tulis sel satu per satu.

Sub ContohArray()
    ' Array statis — ukuran tetap
    Dim bulan(1 To 12) As String
    bulan(1) = "Januari"
    bulan(2) = "Februari"
    ' ... dst

    ' Array dinamis — ukuran fleksibel
    Dim dataNilai() As Double
    Dim n As Long
    n = Cells(Rows.Count, "A").End(xlUp).Row
    ReDim dataNilai(1 To n)

    ' TEKNIK CEPAT: baca semua sekaligus dari range
    Dim arrData As Variant
    arrData = Range("A1:C" & n).Value  ' 2D array

    ' Proses dalam memori (super cepat!)
    Dim i As Long
    For i = 1 To UBound(arrData, 1)
        arrData(i, 3) = arrData(i, 1) * arrData(i, 2)
    Next i

    ' Tulis hasil sekaligus ke sheet (1 operasi!)
    Range("A1:C" & n).Value = arrData
End Sub
🚀Teknik array-in / array-out bisa membuat kode 10-100x lebih cepat dibanding loop baca-tulis sel satu per satu. Sangat direkomendasikan untuk data besar (>1000 baris).
Error Handling
🛡️

On Error & Resume

Menangani kesalahan agar program tidak berhenti tiba-tiba

Lanjut

Error handling mencegah program crash saat terjadi kesalahan. VBA menggunakan On Error GoTo untuk mengarahkan eksekusi ke blok penanganan error.

Sub BukaFile(namaFile As String)
    On Error GoTo TangkapError  ' aktifkan handler

    Dim wb As Workbook
    Set wb = Workbooks.Open(namaFile)

    ' Lanjutkan proses normal...
    MsgBox "File berhasil dibuka!"

    On Error GoTo 0  ' matikan handler, keluar normal
    Exit Sub         ' PENTING: lewati blok error

TangkapError:
    ' Err.Number = kode error, Err.Description = pesannya
    Select Case Err.Number
        Case 1004  ' File tidak ditemukan
            MsgBox "File tidak ditemukan: " & namaFile
        Case 9     ' Subscript out of range
            MsgBox "Sheet tidak ditemukan!"
        Case Else
            MsgBox "Error " & Err.Number & ": " & Err.Description
    End Select

    ' On Error Resume Next — lanjutkan meski ada error
    ' (gunakan dengan hati-hati!)
End Sub
💡Selalu tambahkan Exit Sub sebelum label error agar kode normal tidak masuk ke blok penanganan error. Jangan gunakan On Error Resume Next terlalu luas karena bisa menyembunyikan bug.
