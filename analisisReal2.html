Sifat Aljabar Bilangan Real

Himpunan bilangan real ℝ beserta operasi penjumlahan + dan perkalian · membentuk suatu lapangan (field). Sifat-sifat berikut berlaku untuk setiap a, b, c ∈ ℝ.

Suatu himpunan F dengan operasi biner + dan · disebut lapangan (field) jika memenuhi sepuluh aksioma lapangan yang dibagi menjadi dua kelompok: sifat penjumlahan (A1–A5) dan sifat perkalian (M1–M5).

A. Sifat-Sifat Penjumlahan

A1 · Tertutup Aksioma
a + b ∈ ℝ
Jumlah dua bilangan real adalah bilangan real.
A2 · Komutatif Aksioma
a + b = b + a
Urutan penjumlahan tidak mempengaruhi hasil.
A3 · Asosiatif Aksioma
(a + b) + c = a + (b + c)
Pengelompokan tidak mempengaruhi hasil penjumlahan.
A4 · Elemen Identitas Aksioma
∃ 0 ∈ ℝ: a + 0 = a
Terdapat elemen nol sebagai identitas penjumlahan.
A5 · Invers Penjumlahan Aksioma
∃ (−a) ∈ ℝ: a + (−a) = 0
Setiap bilangan real memiliki invers penjumlahan.

B. Sifat-Sifat Perkalian

M1 · Tertutup Aksioma
a · b ∈ ℝ
Hasil kali dua bilangan real adalah bilangan real.
M2 · Komutatif Aksioma
a · b = b · a
Urutan perkalian tidak mempengaruhi hasil.
M3 · Asosiatif Aksioma
(a · b) · c = a · (b · c)
Pengelompokan tidak mempengaruhi hasil perkalian.
M4 · Elemen Identitas Aksioma
∃ 1 ∈ ℝ, 1≠0: a · 1 = a
Terdapat elemen satu sebagai identitas perkalian.
M5 · Invers Perkalian Aksioma
∀ a≠0, ∃ a⁻¹: a · a⁻¹ = 1
Setiap bilangan real tak-nol memiliki invers perkalian.
D · Distributif Aksioma
a(b + c) = ab + ac
Perkalian distributif terhadap penjumlahan.

C. Teorema-Teorema Aljabar Penting

Dari kesepuluh aksioma lapangan di atas, dapat diturunkan berbagai teorema berikut:

Elemen identitas penjumlahan 0 pada ℝ adalah unik. Artinya, jika terdapat 0' sedemikian sehingga a + 0' = a untuk semua a ∈ ℝ, maka 0' = 0.
Bukti:
Misalkan 0 dan 0' keduanya identitas penjumlahan.
Karena 0 adalah identitas : 0 + 0' = 0'
Karena 0' adalah identitas: 0 + 0' = 0
∴ 0' = 0 (keunikan terbukti) □
Jika a + c = b + c, maka a = b.
Bukti:
a + c = b + c
(a + c) + (−c) = (b + c) + (−c) [tambahkan −c kedua ruas]
a + (c + (−c)) = b + (c + (−c)) [asosiatif A3]
a + 0 = b + 0 [A5: c + (−c) = 0]
a = b [A4: x + 0 = x] □
Untuk setiap a ∈ ℝ, berlaku a · 0 = 0.
Bukti:
a · 0 = a · (0 + 0) [karena 0 + 0 = 0]
= a·0 + a·0 [distributif D]
a·0 + (−a·0) = (a·0 + a·0) + (−a·0)
0 = a·0 + (a·0 + (−a·0)) [asosiatif]
0 = a·0 + 0 = a·0 □
Untuk setiap a ∈ ℝ, berlaku (−1) · a = −a.
Bukti:
a + (−1)·a = 1·a + (−1)·a [M4]
= (1 + (−1))·a [distributif]
= 0·a = 0 [Teorema 3]
∴ (−1)·a adalah invers penjumlahan dari a, yakni −a □
Untuk setiap a, b ∈ ℝ, berlaku (−a)(−b) = ab.
Teorema-teorema di atas tidak diterima begitu saja — melainkan diturunkan secara logis dari aksioma. Inilah hakikat Analisis Real: setiap pernyataan harus dibuktikan dari prinsip yang paling mendasar.

Sifat Urutan Bilangan Real

Selain struktur lapangan, ℝ juga dilengkapi relasi urutan < yang menjadikannya lapangan terurut (ordered field). Sifat urutan memungkinkan kita membandingkan bilangan-bilangan real.

Terdapat himpunan P ⊂ ℝ yang disebut bilangan real positif sedemikian sehingga:
O1 (Trikotomi): ∀ a ∈ ℝ, tepat satu dari: a ∈ P, −a ∈ P, a = 0
O2 (Tertutup-+): a,b ∈ P ⟹ a + b ∈ P
O3 (Tertutup-·): a,b ∈ P ⟹ a · b ∈ P
Kita definisikan: a < b ⟺ b − a ∈ P

A. Aksioma dan Sifat Urutan

SifatPernyataan FormalKeterangan
Trikotomi a<b, a=b, atau a>b Tepat satu berlaku
Transitif a<b dan b<c ⟹ a<c Urutan diwariskan
Penjumlahan a<b ⟹ a+c < b+c Menambah konstanta sama
Perkalian (+) a<b, c>0 ⟹ ac < bc Kali bilangan positif
Perkalian (−) a<b, c<0 ⟹ ac > bc Kali bilangan negatif: tanda berbalik

B. Teorema Urutan Penting

Untuk setiap a ∈ ℝ, berlaku a² ≥ 0. Khususnya, 1 > 0.
Bukti (kasus a ≠ 0):
Kasus 1: a > 0 ⟹ a · a > 0 (O3) ⟹ a² > 0 ✓
Kasus 2: a < 0 ⟹ −a > 0 (def)
⟹ (−a)(−a) > 0 (O3)
⟹ a² > 0 (Teorema 5) ✓
Khusus: 1 = 1² > 0 □
Jika 0 < a < b, maka 0 < 1/b < 1/a.
Untuk setiap a, b ∈ ℝ:
|a + b| ≤ |a| + |b|
Ini adalah salah satu teorema paling fundamental dalam Analisis Real. Juga berlaku:
||a| − |b|| ≤ |a − b| (ketaksamaan segitiga terbalik)
Jika a < b, maka terdapat c ∈ ℝ sedemikian sehingga a < c < b.
Bukti:
Ambil c = (a + b)/2
c − a = (b − a)/2 > 0 (karena b > a) ⟹ c > a
b − c = (b − a)/2 > 0 ⟹ b > c
∴ a < c < b □

C. Nilai Mutlak

Untuk a ∈ ℝ, nilai mutlak didefinisikan:
|a| = ⎧ a, jika a ≥ 0 ⎨ ⎩ −a, jika a < 0

Nilai mutlak memiliki sifat-sifat:

Non-negatif
|a| ≥ 0
dan |a| = 0 ⟺ a = 0
Simetri
|−a| = |a|
Nilai mutlak dan negatifnya sama.
Multiplikatif
|ab| = |a|·|b|
Nilai mutlak perkalian sama dengan perkalian nilai mutlak.
Ketaksamaan Segitiga
|a+b| ≤ |a|+|b|
Teorema paling fundamental analisis real.

Latihan Pembuktian

Berikut adalah latihan pembuktian untuk memperdalam pemahaman Anda terhadap sifat aljabar dan sifat urutan bilangan real. Tulislah pembuktian lengkap menggunakan aksioma-aksioma yang telah dipelajari. Petunjuk tersedia jika dibutuhkan.

Panduan: Setiap pembuktian harus menyebutkan aksioma atau teorema yang digunakan pada setiap langkah. Gunakan notasi: A1–A5 (aksioma penjumlahan), M1–M5 (aksioma perkalian), D (distributif), O1–O3 (aksioma urutan).
Latihan 1 — Tingkat: Dasar

Buktikan: Untuk setiap a ∈ ℝ, berlaku 0 · a = 0.

Petunjuk tersedia jika diperlukan.

Gunakan fakta bahwa 0 = 0 + 0 (aksioma A4), kemudian terapkan sifat distributif. Ingat pula bahwa aksioma urutan M2 berlaku untuk perkalian.
Latihan 2 — Tingkat: Dasar

Buktikan: Untuk setiap a ∈ ℝ, invers penjumlahan −a adalah unik.

Misalkan terdapat b dan c yang keduanya merupakan invers penjumlahan dari a. Gunakan definisi invers (A5) dan hukum pembatalan untuk menunjukkan b = c.
Latihan 3 — Tingkat: Menengah

Buktikan: Jika ab = 0, maka a = 0 atau b = 0. (Tidak ada pembagi nol)

Asumsikan a ≠ 0, maka a⁻¹ ada (M5). Kalikan kedua ruas persamaan ab = 0 dengan a⁻¹ dari kiri, kemudian sederhanakan menggunakan Teorema 3.
Latihan 4 — Tingkat: Menengah

Buktikan: Untuk setiap a, b ∈ ℝ, berlaku −(a + b) = (−a) + (−b).

Tunjukkan bahwa (a + b) + ((−a) + (−b)) = 0 menggunakan aksioma komutatif, asosiatif, dan A5. Dengan keunikan invers, simpulkan hasilnya.
Latihan 5 — Tingkat: Menengah

Buktikan: Jika a < b dan c < d, maka a + c < b + d.

Gunakan sifat penjumlahan pada urutan dua kali: dari a < b tambahkan c, lalu dari c < d tambahkan b. Gunakan transitivitas untuk menggabungkan keduanya.
Latihan 6 — Tingkat: Menengah

Buktikan: Jika 0 < a < 1, maka a² < a.

Dari a < 1 dan a > 0, kalikan kedua ruas dengan a (yang positif). Gunakan sifat perkalian pada urutan.
Latihan 7 — Tingkat: Menengah-Lanjut

Buktikan Ketaksamaan Bernoulli: Untuk x > −1 dan n ∈ ℕ, berlaku (1 + x)ⁿ ≥ 1 + nx.

Gunakan induksi matematika.

Basis: Untuk n=1, persamaan berlaku trivial. Langkah Induksi: Asumsikan benar untuk n=k. Untuk n=k+1, kalikan (1+x)^k ≥ 1+kx dengan (1+x) > 0. Ingat bahwa kx² ≥ 0.
Latihan 8 — Tingkat: Lanjut

Buktikan: Untuk setiap a, b ∈ ℝ, berlaku ketaksamaan segitiga: |a + b| ≤ |a| + |b|.

Perhatikan bahwa −|a| ≤ a ≤ |a| dan −|b| ≤ b ≤ |b|. Tambahkan kedua ketaksamaan ini. Gunakan definisi nilai mutlak untuk menyimpulkan bahwa |a + b| ≤ |a| + |b|. Pisahkan kasus sesuai tanda dari a + b.
Latihan 9 — Tingkat: Lanjut

Buktikan: Untuk setiap a, b ∈ ℝ, berlaku: ||a| − |b|| ≤ |a − b|. (Ketaksamaan Segitiga Terbalik)

Tulis a = (a − b) + b, kemudian terapkan ketaksamaan segitiga (Latihan 8) untuk mendapatkan |a| ≤ |a−b| + |b|. Lakukan hal serupa dengan menukar peran a dan b. Gabungkan kedua hasil tersebut.
Latihan 10 — Tingkat: Tantangan

Buktikan: Tidak terdapat bilangan real terkecil yang positif. Artinya, untuk setiap ε > 0 terdapat δ > 0 dengan δ < ε.

Diberikan ε > 0. Konstruksi eksplisit: ambil δ = ε/2. Gunakan Teorema 7 (sifat invers terurut) dan sifat perkalian pada urutan untuk menunjukkan 0 < δ < ε.
