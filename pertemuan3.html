1 Bilangan Akar dalam \(\mathbb{R}\)

Dalam Analisis Real, konsep akar berkaitan erat dengan sifat kelengkapan (completeness) bilangan real. Setiap bilangan real non-negatif memiliki akar kuadrat yang tunggal dan non-negatif, yang dikenal sebagai akar utama.

Definisi 1.1 — Akar Kuadrat

Untuk setiap \(a \in \mathbb{R},\ a \geq 0\), didefinisikan akar kuadrat utama dari \(a\), dinotasikan \(\sqrt{a}\), sebagai bilangan real \(b \geq 0\) sedemikian sehingga \(b^2 = a\).

Secara formal: \(\sqrt{a} = b \iff b \geq 0 \text{ dan } b^2 = a.\)

Akar ke-\(n\)

Generalisasi akar kuadrat ke akar ke-\(n\) didefinisikan sebagai berikut:

Definisi 1.2 — Akar ke-n

Untuk \(n \in \mathbb{N},\ n \geq 1\), dan \(a \in \mathbb{R}\), akar ke-\(n\) dari \(a\) dinotasikan \(\sqrt[n]{a}\) adalah bilangan \(b\) yang memenuhi \(b^n = a\).

  • Jika \(n\) genap: diperlukan \(a \geq 0\) dan \(b \geq 0\) (akar utama).
  • Jika \(n\) ganjil: \(a\) boleh sembarang bilangan real.
Teorema 1.1
Eksistensi dan Ketunggalan Akar Positif

Untuk setiap \(a > 0\) dan \(n \in \mathbb{N},\ n \geq 1\), terdapat tepat satu bilangan real positif \(b\) sehingga \(b^n = a\).

Bukti (Sketsa):
Existensi: Definisikan \(S = \{x \in \mathbb{R} \mid x > 0,\ x^n < a\}\). Himpunan \(S\) tidak kosong (karena \(\varepsilon = \min(1, a/2) \in S\)) dan terbatas dari atas. Oleh Sifat Kelengkapan \(\mathbb{R}\), \(\sup S\) ada. Misalkan \(b = \sup S\). Dengan menggunakan ketaksamaan binomial dan argumen kontradiksi, dapat ditunjukkan bahwa \(b^n = a\).

Ketunggalan: Jika \(b_1, b_2 > 0\) dan \(b_1^n = b_2^n = a\), maka \(b_1^n - b_2^n = 0\), sehingga \((b_1 - b_2)(b_1^{n-1} + \cdots + b_2^{n-1}) = 0\). Karena \(b_1, b_2 > 0\), faktor kedua positif, maka \(b_1 = b_2\). \(\blacksquare\)
■

Contoh 1.1

Sederhanakan \(\sqrt{48} - 2\sqrt{12} + \sqrt{75}\).

Penyelesaian:

\(\sqrt{48} = 4\sqrt{3},\quad 2\sqrt{12} = 4\sqrt{3},\quad \sqrt{75} = 5\sqrt{3}\)

\(\Rightarrow 4\sqrt{3} - 4\sqrt{3} + 5\sqrt{3} = 5\sqrt{3}\)

Irasionalitas \(\sqrt{2}\)
Teorema 1.2
Irasionalitas Akar Dua

\(\sqrt{2} \notin \mathbb{Q}\). Artinya, \(\sqrt{2}\) adalah bilangan irasional.

Bukti (kontradiksi):
Asumsikan \(\sqrt{2} = \dfrac{p}{q}\) dengan \(p, q \in \mathbb{Z}\), \(q \neq 0\), dan \(\gcd(p,q)=1\).
Maka \(2q^2 = p^2\), sehingga \(p^2\) genap \(\Rightarrow p\) genap. Tulis \(p = 2k\).
Substitusi: \(2q^2 = 4k^2 \Rightarrow q^2 = 2k^2\), sehingga \(q\) genap.
Ini kontradiksi dengan \(\gcd(p,q)=1\). Jadi \(\sqrt{2} \notin \mathbb{Q}\). \(\blacksquare\)

2 Persamaan Kuadrat

Persamaan kuadrat adalah persamaan polinomial berderajat dua dalam satu variabel. Dalam konteks Analisis Real, kajian persamaan kuadrat melibatkan eksistensi akar real, sifat diskriminan, serta hubungan dengan fungsi kuadrat di \(\mathbb{R}\).

Definisi 2.1 — Persamaan Kuadrat

Persamaan kuadrat dalam variabel \(x \in \mathbb{R}\) adalah persamaan berbentuk: \[ax^2 + bx + c = 0\] di mana \(a, b, c \in \mathbb{R}\) dan \(a \neq 0\).

Rumus Kuadratik (Rumus ABC)
\[x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\]
Teorema 2.1
Rumus Kuadratik

Persamaan \(ax^2 + bx + c = 0\) (\(a \neq 0\)) memiliki penyelesaian: \[x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\]

Bukti (melengkapi kuadrat):
\(ax^2 + bx + c = 0\)
\(\Rightarrow x^2 + \dfrac{b}{a}x = -\dfrac{c}{a}\)
\(\Rightarrow x^2 + \dfrac{b}{a}x + \dfrac{b^2}{4a^2} = \dfrac{b^2}{4a^2} - \dfrac{c}{a}\)
\(\Rightarrow \left(x + \dfrac{b}{2a}\right)^2 = \dfrac{b^2 - 4ac}{4a^2}\)
\(\Rightarrow x + \dfrac{b}{2a} = \pm\dfrac{\sqrt{b^2-4ac}}{2a}\)
\(\Rightarrow x = \dfrac{-b \pm \sqrt{b^2-4ac}}{2a} \qquad \blacksquare\)
■
Diskriminan dan Jenis Akar

Diskriminan \(D = b^2 - 4ac\) menentukan sifat akar persamaan kuadrat di \(\mathbb{R}\):

Nilai \(D\)Jenis AkarKeterangan
\(D > 0\)Dua akar real berbeda\(x_1 \neq x_2,\ x_1,x_2 \in \mathbb{R}\)
\(D = 0\)Satu akar real (kembar)\(x_1 = x_2 = -\frac{b}{2a}\)
\(D < 0\)Tidak ada akar realAkar kompleks \(\notin \mathbb{R}\)
Hubungan Akar dan Koefisien (Vieta)
Teorema 2.2 — Rumus Vieta
Hubungan Akar dan Koefisien

Jika \(x_1\) dan \(x_2\) adalah akar-akar dari \(ax^2 + bx + c = 0\), maka: \[x_1 + x_2 = -\frac{b}{a} \qquad \text{dan} \qquad x_1 \cdot x_2 = \frac{c}{a}\]

Bukti:
Dari rumus kuadratik: \(x_1 = \dfrac{-b+\sqrt{D}}{2a}\), \(x_2 = \dfrac{-b-\sqrt{D}}{2a}\)

Jumlah: \(x_1+x_2 = \dfrac{-b+\sqrt{D}-b-\sqrt{D}}{2a} = \dfrac{-2b}{2a} = -\dfrac{b}{a}\)

Hasil kali: \(x_1 \cdot x_2 = \dfrac{(-b)^2 - (\sqrt{D})^2}{4a^2} = \dfrac{b^2-D}{4a^2} = \dfrac{4ac}{4a^2} = \dfrac{c}{a} \qquad \blacksquare\)
■
Bentuk Faktor

Persamaan \(ax^2+bx+c=0\) dapat difaktorkan menjadi:

\[a(x - x_1)(x - x_2) = 0\]
Contoh 2.1

Selesaikan \(2x^2 - 5x + 3 = 0\).

Penyelesaian: \(a=2,\ b=-5,\ c=3\)

\(D = (-5)^2 - 4(2)(3) = 25 - 24 = 1 > 0\)

\(x = \dfrac{5 \pm \sqrt{1}}{4} \Rightarrow x_1 = \dfrac{6}{4} = \dfrac{3}{2},\quad x_2 = \dfrac{4}{4} = 1\)

Verifikasi Vieta: \(x_1+x_2 = \dfrac{5}{2} = -\dfrac{(-5)}{2}\) ✓ dan \(x_1 x_2 = \dfrac{3}{2} = \dfrac{3}{2}\) ✓

3 Nilai Mutlak

Nilai mutlak (modulus) adalah konsep fundamental dalam Analisis Real yang mengukur "jarak" suatu bilangan real dari nol pada garis bilangan. Konsep ini menjadi dasar dari definisi metrik standar pada \(\mathbb{R}\) dan konvergensi barisan bilangan real.

Definisi 3.1 — Nilai Mutlak

Untuk setiap \(x \in \mathbb{R}\), nilai mutlak dari \(x\) didefinisikan sebagai: \[|x| = \begin{cases} x, & \text{jika } x \geq 0 \\ -x, & \text{jika } x < 0 \end{cases}\]

Ekuivalensi: \(|x| = \sqrt{x^2}\) untuk semua \(x \in \mathbb{R}\).

Interpretasi Geometris

Nilai mutlak \(|x|\) menyatakan jarak titik \(x\) dari titik asal pada garis bilangan real. Lebih umum, \(|x - y|\) menyatakan jarak antara titik \(x\) dan \(y\) di \(\mathbb{R}\), menjadikan \(d(x,y) = |x-y|\) sebagai fungsi metrik standar pada \(\mathbb{R}\).

Sifat-Sifat Nilai Mutlak
Teorema 3.1
Sifat-Sifat Dasar Nilai Mutlak

Untuk setiap \(x, y \in \mathbb{R}\), berlaku:

  • (P1) \(|x| \geq 0\) dan \(|x| = 0 \iff x = 0\)  Positivitas
  • (P2) \(|-x| = |x|\)  Simetri
  • (P3) \(|xy| = |x|\cdot|y|\)  Multiplikatif
  • (P4) \(\left|\dfrac{x}{y}\right| = \dfrac{|x|}{|y|}\), untuk \(y \neq 0\)  Pembagian
  • (P5) \(-|x| \leq x \leq |x|\)
  • (P6) \(|x+y| \leq |x|+|y|\)  Ketaksamaan Segitiga
· · · ✦ · · ·
Teorema 3.2 — Teorema Terpenting
Ketaksamaan Segitiga (Triangle Inequality)

Untuk semua \(x, y \in \mathbb{R}\): \[|x + y| \leq |x| + |y|\]

Bukti:
Dari (P5): \(-|x| \leq x \leq |x|\) dan \(-|y| \leq y \leq |y|\).
Menjumlahkan: \(-(|x|+|y|) \leq x+y \leq |x|+|y|\).
Ini ekuivalen dengan \(|x+y| \leq |x|+|y|\). \(\blacksquare\)
■
Teorema 3.3
Ketaksamaan Segitiga Terbalik (Reverse Triangle Inequality)

Untuk semua \(x, y \in \mathbb{R}\): \[\big||x| - |y|\big| \leq |x - y|\]

Bukti:
Dari ketaksamaan segitiga: \(|x| = |(x-y)+y| \leq |x-y|+|y|\), sehingga \(|x|-|y| \leq |x-y|\).
Begitu pula \(|y|-|x| \leq |y-x| = |x-y|\).
Karena \(\big||x|-|y|\big| = \max(|x|-|y|, |y|-|x|)\), maka \(\big||x|-|y|\big| \leq |x-y|\). \(\blacksquare\)
■
Persamaan dan Ketaksamaan Nilai Mutlak
Definisi 3.2 — Penyelesaian Persamaan dan Ketaksamaan

Untuk \(a > 0\):

  • \(|x| = a \iff x = a \text{ atau } x = -a\)
  • \(|x| < a \iff -a < x < a\)
  • \(|x| > a \iff x > a \text{ atau } x < -a\)
  • \(|x| \leq a \iff -a \leq x \leq a\)
  • \(|x| \geq a \iff x \geq a \text{ atau } x \leq -a\)
Contoh 3.1

Selesaikan \(|2x - 3| < 5\).

Penyelesaian:

\(-5 < 2x - 3 < 5\)
\(-2 < 2x < 8\)
\(-1 < x < 4\)
Himpunan penyelesaian: \((-1, 4)\)

Contoh 3.2

Selesaikan \(|x^2 - 4| = 0\).

Penyelesaian: \(|x^2-4|=0 \Rightarrow x^2-4=0 \Rightarrow x = \pm 2\)

4 Kumpulan Teorema Penting

Berikut adalah ringkasan teorema-teorema penting yang menghubungkan ketiga topik dalam konteks Analisis Real:

Teorema 4.1
Nilai Mutlak sebagai Akar Kuadrat

Untuk setiap \(x \in \mathbb{R}\): \(\quad|x| = \sqrt{x^2}\)

Bukti: Jika \(x \geq 0\): \(\sqrt{x^2} = x = |x|\). Jika \(x < 0\): \(\sqrt{x^2} = \sqrt{(-x)^2} = -x = |x|\). \(\blacksquare\)
■
Teorema 4.2
Penyelesaian Persamaan Kuadrat dengan Nilai Mutlak

Persamaan \(ax^2+bx+c=0\) memiliki akar real genap-genap \(\iff D = b^2-4ac \geq 0\), dan:

\[|x_1 - x_2| = \frac{\sqrt{D}}{|a|}\]

Bukti: \(x_1 - x_2 = \dfrac{\sqrt{D}}{a}\) (atau \(-\dfrac{\sqrt{D}}{a}\)), sehingga \(|x_1-x_2| = \dfrac{\sqrt{D}}{|a|}\). \(\blacksquare\)
■
Teorema 4.3
Cauchy–Schwarz untuk \(\mathbb{R}^2\)

Untuk \(a_1,a_2,b_1,b_2 \in \mathbb{R}\):

\[|a_1 b_1 + a_2 b_2| \leq \sqrt{a_1^2+a_2^2}\cdot\sqrt{b_1^2+b_2^2}\]

Teorema 4.4
AM–GM (Rata-Rata Aritmetik ≥ Rata-Rata Geometrik)

Untuk \(a, b \geq 0\):

\[\frac{a+b}{2} \geq \sqrt{ab}\]

dengan kesamaan terjadi jika dan hanya jika \(a = b\).

Bukti: Perhatikan bahwa \((\sqrt{a}-\sqrt{b})^2 \geq 0\), sehingga \(a - 2\sqrt{ab} + b \geq 0\), yaitu \(a+b \geq 2\sqrt{ab}\). Kesamaan terjadi \(\iff \sqrt{a} = \sqrt{b} \iff a = b\). \(\blacksquare\)
■
Teorema 4.5
Lemma Ketaksamaan Berguna

Untuk semua \(x, y \in \mathbb{R}\) dan \(\varepsilon > 0\):

\[|xy| \leq \frac{\varepsilon}{2}x^2 + \frac{1}{2\varepsilon}y^2\]

(Young's Inequality, berguna dalam pembuktian konvergensi)

Teorema 4.6
Sifat Archimedean dan Akar

Untuk setiap \(\varepsilon > 0\), terdapat \(n \in \mathbb{N}\) sehingga \(\dfrac{1}{\sqrt{n}} < \varepsilon\).

Bukti: Dari Sifat Archimedean, untuk \(\varepsilon^2>0\) terdapat \(n\in\mathbb{N}\) dengan \(\dfrac{1}{n}<\varepsilon^2\), sehingga \(\dfrac{1}{\sqrt{n}}<\varepsilon\). \(\blacksquare\)
■
