Bagian I — Akar Kuadrat
1
Akar Kuadrat
Buktikan bahwa akar kuadrat dari 2 adalah bilangan irasional.
Buktikan: √2 ∉ ℚ (Gunakan metode kontradiksi. Asumsikan √2 = p/q dengan p, q ∈ ℤ, q ≠ 0, dan (p,q) = 1.)
Ruang Pembuktian
2
Akar Kuadrat
Buktikan bahwa fungsi akar kuadrat kontinu di setiap titik pada domain alaminya.
Buktikan: f(x) = √x kontinu di setiap x₀ > 0 Artinya: ∀ε > 0, ∃δ > 0 sehingga |x - x₀| < δ ⟹ |√x - √x₀| < ε
Ruang Pembuktian
3
Akar Kuadrat
Buktikan ketidaksamaan berikut menggunakan sifat akar kuadrat.
Buktikan: ∀a, b ≥ 0, √(ab) ≤ (a + b) / 2 (Ini adalah ketidaksamaan AM–GM untuk dua variabel. Mulailah dari fakta bahwa (√a − √b)² ≥ 0.)
Ruang Pembuktian
4
Akar Kuadrat
Buktikan konvergensi barisan berikut dan tentukan limitnya.
Diberikan barisan aₙ = √(n+1) − √n, n ∈ ℕ. Buktikan bahwa lim(n→∞) aₙ = 0. (Rasionalisasi pembilang mungkin berguna.)
Ruang Pembuktian
Bagian II — Kuadrat
5
Kuadrat
Buktikan bahwa kuadrat dari setiap bilangan real adalah non-negatif, dan berikan karakterisasi kapan kuadrat bernilai nol.
Buktikan: ∀x ∈ ℝ, x² ≥ 0 Buktikan pula: x² = 0 ⟺ x = 0 (Gunakan aksioma urutan pada ℝ dan sifat perkalian bilangan real.)
Ruang Pembuktian
6
Kuadrat
Buktikan identitas polarisasi berikut yang menghubungkan perkalian dengan kuadrat.
Buktikan: ∀x, y ∈ ℝ, xy = [(x+y)² − (x−y)²] / 4 (Ekspansi aljabar saja tidak cukup — berikan argumen yang menunjukkan struktur identitas ini.)
Ruang Pembuktian
7
Kuadrat
Buktikan ketidaksamaan Cauchy–Schwarz untuk dua pasang bilangan real.
Buktikan: ∀a, b, c, d ∈ ℝ, (ac + bd)² ≤ (a² + b²)(c² + d²) (Tentukan pula kapan kesamaan berlaku. Pertimbangkan fungsi kuadrat f(t) = (at+c)² + (bt+d)².)
Ruang Pembuktian
Bagian III — Nilai Mutlak
8
Nilai Mutlak
Buktikan Ketidaksamaan Segitiga beserta variasinya.
Buktikan: ∀x, y ∈ ℝ, (i) |x + y| ≤ |x| + |y| (ii) ||x| − |y|| ≤ |x − y| (Untuk (ii), tunjukkan bahwa (ii) merupakan konsekuensi langsung dari (i).)
Ruang Pembuktian
9
Nilai Mutlak
Buktikan ekuivalensi antara nilai mutlak dan akar kuadrat dari kuadrat.
Buktikan: ∀x ∈ ℝ, |x| = √(x²) Kemudian gunakan fakta ini untuk membuktikan: ∀x, y ∈ ℝ, |xy| = |x| · |y|
Ruang Pembuktian
10
Nilai Mutlak
Buktikan teorema karakterisasi keterbatasan barisan menggunakan nilai mutlak.
Diberikan barisan (xₙ) di ℝ. Buktikan: (xₙ) konvergen ke L ⟺ ∀ε > 0, ∃N ∈ ℕ sehingga ∀n ≥ N: |xₙ − L| < ε Selanjutnya, buktikan bahwa limit konvergen bersifat tunggal: Jika xₙ → L dan xₙ → M, maka L = M (Gunakan ketidaksamaan segitiga dan argument ε/2.)
Ruang Pembuktian
